# App-wide Workspace Stack

Fluidity persists an app-wide Workspace Stack of Open Workspace IDs ordered by most-recently-shown, with the first entry serving as the Current Workspace in the current single-window app. This replaces timestamp-based workspace selection because Open Workspaces are parallel working environments rather than stopped sessions, and future workspace switching shortcuts, Workspace Switcher Tiles, and multi-window support need an explicit ordering model.
